Software I Use

I don't like slow and complicated things. These are the the programs I think which are simple and fast enough to get something done quickly without having to stress over them.

Operating System/Distribution

Artix Linux

artix is a fork of arch linux without systemd. I don't have a very powerful computer. systemd became a pain in some point i was searching something less bloated & i found artix. Using it for more than 1 year and still loving it.

Terminal

Suckless Terminal (ST)

st is a very mininal and high customizable terminal for x. i didn't configure it myself i use siduck's build of it with few twerks. this is all i need to get most things done.

Shell

fish

fish is the easiest shell to use. It's just plain english nothing complicated which is why I'm in love with it. I have some custom functions to do somethings but that's all.

Window Manager/Desktop Environment

Suckless Window Manager (DWM)

dwm is a window manager for x that focuses on simplicity. I use archcraft os's build for this with a custom configuration. simply installing the package won't work becuse it's configured to work with lots of other things be sure to read the config and figure out what's going on.

Text editing and programming

NeoVim

vim is the goto text editor of nerds (maybe i'm one idk). neovim is a extended custom version of vim which has lot more features & flexibility. I use NvChad as my base configuration. I have added some LSP configs to it make it more suitable to code.

Web browser

LibreWolf

librewolf is a firefox fork without all those trackers. it has some pretty good privacy features. I'm using it Eric's userChrome theme with it.

Utilities

File manager

LF

LF is a replacement for Ranger which i used in the past. It's written in Go and fast enough to satify me. I have some custom configuration top of the base config & some nice icons.

Mail client

aerc

aerc is the most simple email client for cli. I've been using it for more than 6 months now, It can get most of the work done. sometimes i switch over to thunderbird for some gpg stuff.

Music/audio player

mpd & ncmpcpp

most of my music is available locally. So i just use this as my music player. I also use archcraft os's configuration for this. you can do many things with mpd.

Video player

MPV

this is the only player you should be using. you can fine tune this to suite your eyes perfectly. all the cool kids are using it.

RSS reader

NewsBoat

I don't wanna log into sites just to check whether if there's new content. So newsboat is my saviorrrrr. Rss is a wonderful thing. yall should use it.

Password manager

keepassxc

everyone should use a password manager. i recommend keepassxc. It's very flexible and simple. It can also intergrate with the browser without compromising anything.

Torrent client

rTorrent

most people find rtorrent very complicated but it's actually not that complicated. I using it with some default settings it's been doing great so far. I used to have rtorrent-ps which is not maintained anymore. It looked more geeky.

PDF viewer

zathura

zathura is perfect just to view docments there's nothing fancy which is why I love it.

Image viewer

nsxiv

nsxiv is a image viewer for x. which adds some optimizations and flexibility over the original sxiv.

Remote desktop client

remmina

remmina is the best choise if you wanna manage so many remote desktops. i can work with few popular protocols.

Cloud storage manager

rclone

everyone knows rclone. It's the most famous tool out there to manage diffrent cloud storage stuff. It can do many things.

Container manager

podman

podman is a alternative to docker which can do almost everything that docker can + more cool stuff like rootless containers.

Production

Video and Audio

ffmpeg & kdenlive

most of the time ffmpeg can get the work done. sometimes i switch over to kdenlive to just make some simple changes. I don't like video editing now so I always try to do things with ffmpeg instead of kdenlive.

Image modification

imagemagick & gimp

i rarely use gimp. but i do use it sometimes just to draw some scribbles and to crop some photoes. most of the time it's just converting photos with imagemagick.
